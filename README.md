# Autonomous Navigation of Robot using Reinforcement Learning

## Overview
This repository contains the source code, simulation environments,
trained models, and experimental artifacts used in the paper:

"­­Comparative Analysis of Proximal Policy Optimization and Deep Q-Network for Autonomous Mobile Robot Navigation "

## Repository Structure

src/             - Source code and ROS2 packages
worlds/          - Gazebo simulation environments
models/          - Trained RL models
logs/            - Training and evaluation logs
result_figures/  - Figures used in the paper
docs/            - Experiment documentation

## Requirements

- Ubuntu 24.04
- ROS2 Humble
- Gazebo Harmonic
- Python 3.10+
